What is Html

Html

Html: Stand for hyper text markup.Structures the content of a web page.
CSS: Stand for Cascading Style sheet Styles and designs the appearance of a web page.
JavaScript : Adds interactivity and dynamic behavior to a web page.
It uses Tags to organize such as :
  • Headings
  • Paragraphs
  • Image
  • Links
  • Tabels

Key Features

  1. Uses simple tags which are easy to write and understand .

  3. Html allows you to connect one page to an other using links
